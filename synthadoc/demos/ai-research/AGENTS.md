# AI Research Tracker — Agent Guidelines

## Ingest Guidelines

- Focus on AI/ML research: architectures, training methods, benchmarks, key papers, and researchers
- Summarize key claims, methods, and results — prioritize technical precision over breadth
- Cross-reference related concepts using [[page-name]] link syntax (lowercase, hyphenated slugs)
- When a source updates or refines a prior result, flag contradictions rather than silently overwriting
- Skip sources that are purely promotional, news aggregators, or outside AI/ML research scope

## Query Guidelines

- Answer using only wiki content — never hallucinate facts not present in pages
- Always cite sources using [[page-name]] link syntax
- When a query spans multiple topics, decompose into sub-queries and synthesize a merged answer
- If the wiki does not contain sufficient information, report a knowledge gap and suggest search queries to fill it
